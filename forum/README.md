Forum Project

This project is a forum application built with Next.js, TypeScript, and localStorage, developed as part of a school project. It allows users to create threads, comment on them, search threads, and provides administrative functionalities. The project emphasizes simplicity, a clean user interface, and practical moderation tools.

Features:

Create Threads: Users can create new threads.

Comment System: 
Users can comment on threads, with a structure similar to YouTube where the latest comment appears at the top. Replies are supported and are visually smaller to indicate they are child comments.

Search Function: 
A search bar allows users to filter threads based on keywords.
Profile Management: Each user has a profile page where they can upload a profile picture and write an "About Me" description.

Admin Controls:
Lock Threads: Admins can lock threads to prevent further comments.

User Promotion/Demotion: Admins can promote other users to moderators and demote them.

First Registered User Becomes Admin: The very first user to register is automatically assigned as the admin.

Responsive Design: 
The forum is built with a mobile-first approach, ensuring a seamless experience across devices.

Theme Toggle: 
Users can switch between light and dark modes for a personalized experience.

Getting Started
Clone the repository to your local machine.
Navigate to the project directory.
Install the dependencies using your preferred package manager.
Run the development server to view the project locally.
Open your browser and navigate to the provided local server address.
